Para iniciar a api, importar no Eclipse, executar o mvn clean/install e executar como Java Application.

Para executar os testes, executar a classe PremioApplicationTests como JUnit Test.

O arquivo deve estar em user.home. O nome do arquivo como "movielist.csv". Ao iniciar a aplicação é feito o sysout do caminho.

Para fazer os testes, o endpoint é http://localhost:8080/premio/. Pode ser usado o postman ou, se preferir, o swagger: http://localhost:8080/swagger-ui.html.

Caso os endpoints não estejam disponíveis, verificar no console quando foi feita a inicialização em qual porta o serviço está escutando.
